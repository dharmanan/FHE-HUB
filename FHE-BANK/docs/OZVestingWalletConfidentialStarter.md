# OZVestingWalletConfidentialStarter

Minimal deployable vesting wallet using VestingWalletConfidential.

Category: Oz Confidential

## Run Locally

```bash
# Generate a standalone repo for this example
npm run create-example create oz-confidential-vesting-wallet ./output/oz-confidential-vesting-wallet

cd ./output/oz-confidential-vesting-wallet
npm install
npm test
```

Tip: to verify everything in one go, run the hub smoke test:

```bash
npm run smoke-test -- --only oz-confidential-vesting-wallet
```
