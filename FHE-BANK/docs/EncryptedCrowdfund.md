# EncryptedCrowdfund

Crowdfunding with private contributions

Category: Fundraising

## Run Locally

```bash
# Generate a standalone repo for this example
npm run create-example create encrypted-crowdfund ./output/encrypted-crowdfund

cd ./output/encrypted-crowdfund
npm install
npm test
```

Tip: to verify everything in one go, run the hub smoke test:

```bash
npm run smoke-test -- --only encrypted-crowdfund
```
