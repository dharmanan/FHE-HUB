# EncryptedRaffle

Raffle system with encrypted entries

Category: Gaming

## Run Locally

```bash
# Generate a standalone repo for this example
npm run create-example create encrypted-raffle ./output/encrypted-raffle

cd ./output/encrypted-raffle
npm install
npm test
```

Tip: to verify everything in one go, run the hub smoke test:

```bash
npm run smoke-test -- --only encrypted-raffle
```
